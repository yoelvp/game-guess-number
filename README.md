# Gussing game
Game guess the random number.


## Installation
1. Clone repository:
```bash
git clone https://github.com/yoelvp/game-guess-number.git
```

2. Install dependencies:
```bash
cargo build
```

3. Start the game:
```bash
cargo run
```

## Usage
- Enter a random number less than 20.
- You have 3 attempts.
